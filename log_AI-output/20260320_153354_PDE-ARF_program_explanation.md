## PDE-ARF プログラムの説明

### プログラム概要

**PDE-ARF**（`main.py`）は、東京電機大学の**新聞委員会**が大学自治会に提出する「**会計要望書**」をGUIで簡単に作成するデスクトップアプリケーションです。

`customtkinter`（モダンなTkinterラッパー）を使用したGUIアプリで、Wordテンプレートファイル（`PDM_ARF-001.docx`）に入力内容を書き込んで `.docx` 形式で保存します。

---

### 使用ライブラリ

| ライブラリ | 役割 |
|---|---|
| `tkinter` / `customtkinter` | GUIウィンドウの構築 |
| `docx` | Wordファイルの読み書き |
| `tkinter.filedialog` | 保存ダイアログの表示 |
| `time` | 保存ファイル名への日付付与 |
| `os` | 作成したWordファイルを自動で開く |

---

### 動作モード一覧

ウィンドウ下部のボタンで**3つのモード**を切り替えます。

| モード (`flag_mode`) | ボタン名 | 要望書の種類 |
|---|---|---|
| `0` | **汎用** | 物品の購入・使用（カメラ等の機材） |
| `1` | **交通費（電車・バス）** | 電車・バスの交通費申請 |
| `2` | **交通費（レンタカー）** | レンタカーの交通費申請 |

---

### 主要な関数の役割

#### `button_enter()` — メインロジック
入力内容をWordテンプレートに書き込む中心的な関数です。

1. `flag_use`（コンボボックスの選択値）を確認し、「購入」か「使用」かを判定
2. `PDM_ARF-001.docx` を読み込み、各段落に値を上書き
3. **共通項目**（日付・団体名・代表者名・会計担当者名・チェックボックス・要望内容）を書き込み
4. **モード別の固有項目**を書き込み（品名・金額・日付・経路など）
5. ファイル保存ダイアログでパスを指定し `.docx` 保存
6. 保存したファイルを `os.startfile()` で自動的に開いてプログラム終了

#### `button_change1/2/3()` — モード切替
- 選択されたモードの入力フレームを `grid()` / `grid_forget()` で表示・非表示
- チェックボックスの有効/無効を切り替え（交通費モード時はチェックボックスが固定）

#### `combo_select()` — コンボボックス整合性チェック
4つのコンボボックス（品名・金額・日付・理由）がすべて「購入」または「使用」に揃っているかを検証。揃っていない場合は `flag_use = 0` となりエラーを表示。

#### `button_end()` — 終了確認
確認ダイアログを出してからウィンドウを閉じます。

#### `mouse_y_scroll()` — スクロール
マウスホイールでCanvasをスクロールできるようにする補助関数。

### GUI構造

```
root (CTk ウィンドウ 800×800)
└── frame (スクロール領域の外枠)
    ├── canvas (スクロール可能エリア)
    │   └── scrollable_frame
    │       ├── la_title (タイトルラベル「会計要望書生成」)
    │       ├── fr_body (共通入力エリア)
    │       │   ├── 日付・団体名・代表者名・会計担当者名の入力欄
    │       │   ├── チェックボックス7個（用件選択）
    │       │   └── 要望内容テキストボックス
    │       ├── fr_mode1 (汎用モード専用入力)
    │       ├── fr_mode2 (電車・バスモード専用入力)
    │       └── fr_mode3 (レンタカーモード専用入力)
    └── fr_button (操作ボタン群)
        ├── 汎用ボタン
        ├── 交通費(電車・バス)ボタン
        ├── 交通費(レンタカー)ボタン
        ├── 入力ボタン（Word生成&保存）
        └── 終了ボタン
```

---

### 処理の流れ（ユーザー操作）

```
起動
 → モードを選択（汎用 / 電車・バス / レンタカー）
 → 必要事項を入力
 → 「入力」ボタンを押す
 → Word (.docx) の保存ダイアログ
 → 保存 → Wordが自動で開く → アプリ終了
```
